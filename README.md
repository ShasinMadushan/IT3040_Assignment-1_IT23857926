# IT3040 - Assignment 1 - Test Automation

**Student Registration ID:** IT23857926
**Subject:** IT3040 - ITPM  
**Target Application:** [SwiftTranslator](https://www.swifttranslator.com/)

## Description
This repository contains automated functional and UI tests for the Swift Translator web application using Playwright.


### Prerequisites
- Node.js (v18+)
- npm

### Installation
npm install

### Run all tests
npx playwright test

### Run tests with UI
npx playwright test --headed
